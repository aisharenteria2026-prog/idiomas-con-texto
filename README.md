# 🌍 Idiomas con Texto

Detector, traductor y lector de idiomas con interfaz gráfica en Python.

![Python](https://img.shields.io/badge/Python-3.8%2B-blue)
![License](https://img.shields.io/badge/License-MIT-yellow)

## ✨ Funcionalidades

- **🔍 Detección automática** de idioma usando `langdetect`
- **🌐 Traducción** a cualquier idioma disponible en Google Translate
- **📍 País y continente** de origen del idioma detectado
- **🗺️ Ver idiomas** por continente (Europa, América, África)
- **🔊 Audio con voz masculina** que lee el texto en voz alta
- **💾 Guardar textos** favoritos con fecha y hora

## 🎨 Colores

Interfaz con diseño en **blanco, azul y amarillo**.

## 🚀 Instalación

```bash
git clone https://github.com/aisharenteria2026-prog/idiomas-con-texto.git
cd idiomas-con-texto
pip install -r requirements.txt
python idiomas_con_texto.py
```

## 📦 Dependencias

- `langdetect` — Detección de idioma
- `deep-translator` — Traducción automática
- `pyttsx3` — Texto a voz

## 📝 Cómo usar

1. Escribe o pega un texto en el cuadro de entrada
2. Selecciona el idioma de destino en el menú desplegable
3. Haz clic en **Detectar y Traducir**
4. Usa **Escuchar** para oír el texto
5. Guarda tus textos favoritos con **Guardar Texto**
